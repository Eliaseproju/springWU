Nirobet
Ett webbaserat spelgränssnitt byggt med HTML, CSS och JavaScript. Projektet innehåller en startsida, en sida för kampanjer samt en plånbokssida.
Funktioner

Startsida: Hero-sektion, livevinstvisning, spelkatalog, sportmatcher med odds och kampanjkort
Kampanjer: Lista över aktiva bonusar och erbjudanden med kopierbar referenslänk
Plånbok: Insättning och uttag via krypto eller kort, samt transaktionshistorik
Sidebar: Hopfällbar navigering med hamburgerknapp som fungerar på både desktop och mobil
Responsiv design: Anpassad layout för både dator och mindre skärmar

Tekniker

HTML5
CSS3
Vanilla JavaScript
